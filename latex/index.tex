\hypertarget{index_autotoc_md5}{}\doxysection{\texorpdfstring{📖 Uso y Ejecución}{📖 Uso y Ejecución}}\label{index_autotoc_md5}

\begin{DoxyEnumerate}
\item {\bfseries{Scraping de Datos\+:}} ~\newline
 Ejecuta el script de scraping (ubicado en {\ttfamily tools/fungi-\/scraping/}) para obtener y almacenar la información de hongos en la base de datos.
\item {\bfseries{Servidor Web\+:}} ~\newline
 Configura tu servidor web (por ejemplo, Apache o Nginx) para apuntar a la carpeta {\ttfamily public} como directorio raíz.
\item {\bfseries{Acceso a la Aplicación\+:}} ~\newline

\begin{DoxyItemize}
\item Navega a {\ttfamily /} para ver el listado de fungis.
\item Accede a {\ttfamily /login} para iniciar sesión.
\item Utiliza las rutas {\ttfamily /admin}, {\ttfamily /profile}, etc., para acceder a las áreas protegidas según corresponda.
\end{DoxyItemize}
\item {\bfseries{Internacionalización\+:}} ~\newline
 La aplicación soporta múltiples idiomas (ej. español e inglés) mediante gettext. Revisa la carpeta {\ttfamily locales} para ver los archivos de traducción.
\end{DoxyEnumerate}\hypertarget{index_autotoc_md6}{}\doxysection{\texorpdfstring{📝 Documentación Adicional}{📝 Documentación Adicional}}\label{index_autotoc_md6}

\begin{DoxyItemize}
\item {\bfseries{Estructura SQL\+:}} ~\newline
 El modelo de datos se encuentra en {\ttfamily src/db/structure.\+sql}. Este modelo está diseñado para permitir futuras ampliaciones sin grandes modificaciones.
\item {\bfseries{Gestión de Sesiones y Cookies\+:}} ~\newline
 La autenticación se maneja mediante JWT, asegurando que los usuarios se mantengan conectados y que las rutas sensibles estén protegidas.
