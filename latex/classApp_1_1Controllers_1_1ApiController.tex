\doxysubsection{Detailed Description}
Controlador para manejar las solicitudes de la API RESTful. 

Esta clase proporciona endpoints para acceder y manipular datos de hongos y usuarios a través de una interfaz API RESTful, implementando métodos para las operaciones CRUD.
