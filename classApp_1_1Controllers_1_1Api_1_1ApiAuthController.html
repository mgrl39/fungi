App\Controllers\Api\ApiAuthController::generateJwtToken ( array  $user)

Genera un token JWT para un usuario.

Parámetros
array$userDatos del usuario
Devuelve
string Token JWT generado

El token JWT generado contiene:

  • Header: typ=JWT, alg=HS256
  • Payload:
    • iat: Timestamp de emisión
    • exp: Timestamp de expiración (1 hora)
    • sub: ID del usuario
    • username: Nombre de usuario
    • role: Rol del usuario
  • Firma: HMAC SHA256 del header y payload codificados
